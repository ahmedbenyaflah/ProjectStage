# Assets folder

Place your app **logo/icon** here so it appears in the header.

- **Filename:** `logo.png` (or `logo.svg` — if you use SVG, update the `<img src>` in `SearchPage.js` to `/assets/logo.svg`).
- **Usage:** The header loads `/assets/logo.png` and shows it to the left of "ORANGE Email Log Viewer". If the file is missing, the logo is hidden automatically.
